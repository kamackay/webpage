Privacy Policy

What information is available to us?

Jeeves has access to:
  • Contacts Used to get the main Google account used on the device
  • Location  Used to keep a log of location at latitude/longitude coordinates (which is an optional feature that can be turned off on the "Features" page)
  • Phone No longer used, but is still in the app manifest temporarily, will be removed the next time the developer (me) has a spare minute
  • Storage Used to store the log file data

What do we do with this information?

The information is stored on your device and never sent to a backend server (we don't even have one). You can view it, delete it, or revoke permissions to it with impunity
